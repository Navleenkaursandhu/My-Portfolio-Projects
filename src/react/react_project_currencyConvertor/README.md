# CURRENCY CONVERTER
Get a fast and easy calculator for converting 100+ currencies using the latest live exchange rates by just selecting correct pair of ISO currency codes. 

Exchange rates keep on fluctuating, and this live Currency Converter fetches the latest exchange rate from an external API making it the ideal tool to keep your eye on the market rate for any given currency. 

You don't even need an account to use this App, just select the currency pair you want to see and the built-in market Exchange Rate Calculator will provide you the latest market values.
